# Spring Boot个人博客项目随笔


**技术组合：**

*  后端：Spring Boot + mybatis 
*  数据库：MySQL
*  前端UI：Semantic UI框架 + thymeleaf模板

**工具与环境：**

*  IDEA
*  Maven 3.6.2
*  JDK 12
*  springboot 2.2.5
